1. Introduction (0:00:00) Bagian ini memperkenalkan DOM (Document Object Model) sebagai inti dari interaksi antara HTML, CSS, dan JavaScript. DOM adalah representasi berbasis objek dari dokumen HTML yang memungkinkan pengembang untuk memanipulasi elemen, gaya, dan konten halaman web secara dinamis.
